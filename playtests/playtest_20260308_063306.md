## New Observations

### Defeat Message Lacks Context
The game over message "Humanity has fallen. Too many lives were lost." is generic. It doesn't show the death toll, the threshold, or any stats. A player seeing this for the first time doesn't know what the threshold was or how close they were. Something like "787M dead (10% of world population)" would give closure.

### Threats Panel Shows Full Region Data at 0% Knowledge
At defeat (tick 891, no player interaction), both diseases are still "Unknown Pathogen" at 0% knowledge. However, the Threats panel shows exact per-region infected/immune/dead counts for the selected disease. The name is hidden ("???") but all numerical data is fully visible. This undermines the knowledge-gating system — the player gets precise epidemiological data for free without any identification research.

### Game is Unwinnable Without Player Interaction
Running the default game to its natural conclusion always results in defeat around tick 891. The diseases grow exponentially and hit the 10% death threshold before burning out. Winning requires active player intervention (research, medicine development, deployment), which is good game design — but the available tools (10K-50K dose medicines) may still be too weak to change the outcome significantly.
