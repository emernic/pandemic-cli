# Naming Style

Read this before naming or renaming player-facing things.

Read [setting.md](/home/emernic/pandemic_cli_8/docs/setting.md) and [framing-guide.md](/home/emernic/pandemic_cli_8/docs/framing-guide.md) first.

This document is about names, not prose in general.

## Core Rule

Name things the way the in-world actor who coined them would name them.

Usually that means:

- N.W.H.O. administrative language
- boardroom / legal / contract language
- actual scientific or technical language
- plain operational language from governors, staff, and field teams

Do not switch into cinematic sci-fi naming, slogan naming, or writerly symbolic naming.

## Salt The Name Pool

LLMs pick the same names over and over. Do not trust your first instinct for names of people, corporations, governors, regions, or institutions.

This is not a stylistic preference. If you skip this step, you will ship obvious default-LLM names. Those names repeat across unrelated sessions and immediately make the writing feel AI-generated.

If you need a proper name, you MUST use web search first.

Do this literally:

1. Use the web search tool.
2. Search for raw lists of real names from the relevant region or a nearby one.
3. Search for raw lists of real company names if you need corporation naming texture.
4. Pick from those lists, or adapt from those lists.
5. Only then write the in-game name.

Good search patterns:

- `common surnames in kazakhstan`
- `common surnames in brazil`
- `list of surnames in nigeria`
- `major companies in azerbaijan`
- `top companies in indonesia`
- `common female names in argentina`

Do NOT just make up a director surname, governor surname, or corporation name from memory. Do NOT trust your first instinct. Do NOT accept names that feel like default LLM authority names.

If a name feels like generic sci-fi management slop, throw it out and search again.

## Use Actual Science

If you are naming a technology, method, platform, assay, therapy, or intervention, you MUST anchor it to real science.

This is not about sounding smart. If you skip this step, you will produce fake biotech wording that sounds plausible only to people who do not know the field. To anyone who does know it, it reads as instant AI slop.

Do this literally:

1. Stop and ask what the thing actually is.
2. Use the real term if a real term exists.
3. If the exact real term does not fit, build from real mechanism, real biology, and real scientific language.
4. Prefer the specific technique or mechanism over the broad fancy-sounding umbrella term.

Do NOT do any of these:

- science-y noise
- made-up platform names
- words chosen for mouth feel rather than accuracy
- vague "advanced biotech" phrasing that sounds impressive but means nothing

Do this instead:

- actual technique
- actual mechanism
- actual scientific field language
- actual assay / process / delivery / platform terminology if that is what the feature really is

The player does not need the name to sound advanced. The name needs to sound like someone who knows what they are talking about wrote it.

## Keep Labels Plain

Most labels should be plainer than your instinct wants.

Good categories for names in this game:

- administrative
- contractual
- technical
- operational
- cold euphemistic, in a real institutional way

Bad categories for names in this game:

- legendary
- prophetic
- heroic
- thriller-coded
- winkingly satirical

If the situation is extreme, let the situation carry that. Do not make the label carry it too.

## Quick Checks

Before keeping a name, ask:

- Who would actually use this term?
- Did I use web search and pull from real raw name lists, or did I let the model pick a default name?
- Is this a real scientific term or built from real scientific logic?
- Is the label identifying a thing, or trying to perform mood?
- If I replaced the coolest word with a plain one, would the name improve?

If yes, replace it.

## Preferred Bias

When in doubt, bias toward:

1. the plain institutional name
2. the real technical name
3. the real legal / commercial name

Bias away from:

1. codename energy
2. faux-prestige sci-fi
3. made-up smart-sounding biotech
4. default LLM proper names
