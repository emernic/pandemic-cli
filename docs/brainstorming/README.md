# Brainstorming Docs

These documents are early design notes.

They are not implemented, not approved, and not a roadmap.

Use them to preserve promising directions and discuss them concretely before anyone starts building.

Rules for this folder:

- Treat every idea here as draft material that still needs iteration, pruning, and play-feel validation.
- Do not treat listed mechanics as requirements.
- Prefer a few strong game ideas over many small subsystems.
- If a concept graduates from brainstorming into real direction, move it into a more specific design doc elsewhere in `docs/`.
